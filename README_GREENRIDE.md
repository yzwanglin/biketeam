# 绿行积分平台 - 使用指南

## 🎉 项目已完成！

绿行积分平台是一个基于绿色骑行的积分奖励与抵扣平台，包含用户端 APP 和公司管理后台。

---

## 📱 平台组成

### 1. 用户端 (user.html)
普通用户使用的客户端，功能包括：
- 🏠 首页 - 数据统计、热门商品
- 🚴 骑行 - 记录骑行里程、获得积分
- 🛒 商城 - 浏览商品、购买、积分抵扣
- 👤 我的 - 个人信息、积分明细

### 2. 管理后台 (admin.html)

- 📊 数据概览 - 用户数、骑行里程、积分发放、成交额
- 👥 用户管理 - 用户列表、查看详情、数据导出
- 📦 商品管理 - 添加/编辑/下架商品、库存管理
- 🛒 订单管理 - 订单列表、状态管理、发货
- 🚴 骑行数据 - 骑行记录查询、数据导出

## 💰 积分规则

### 积分获取
| 行为 | 积分奖励 |
|------|----------|
| 骑行 1 公里 | +10 积分 |
| 完成首次骑行 | +100 积分（新手奖励）|

### 积分使用
**20% 现金抵扣规则：**
- 商品原价 × 20% = 可抵扣现金金额
- 所需积分 = 抵扣金额 ÷ 0.02（100 积分 = 2 元）
- 实际支付 = 商品原价 × 80%

**示例：**
```
商品：骑行头盔 ¥128
可抵扣：¥128 × 20% = ¥25.6
所需积分：¥25.6 ÷ 0.02 = 1280 积分
实际支付：¥128 × 80% = ¥102.4
```

### 积分等级
| 等级 | 积分要求 | 特权 |
|------|----------|------|
| 青铜骑士 | 0-999 | 基础权益 |
| 白银骑士 | 1000-4999 | 积分 1.1 倍 |
| 黄金骑士 | 5000-19999 | 积分 1.2 倍 + 专属客服 |
| 铂金骑士 | 20000-99999 | 积分 1.3 倍 + 优先发货 |
| 钻石骑士 | 100000+ | 积分 1.5 倍 + 专属活动 |

## 🎯 核心功能说明

### 骑行记录
1. 进入"骑行"页面
2. 点击"开始骑行"按钮
3. 系统模拟骑行，实时更新里程和时长
4. 点击"结束骑行"，自动计算积分
5. 积分规则：里程（公里）× 10 = 获得积分

### 商城购物
1. 进入"商城"页面
2. 浏览商品，点击"立即购买"
3. 系统自动计算：
   - 商品价格
   - 可抵扣积分（价格×20%÷0.02）
   - 实际支付金额（价格×80%）
4. 确认购买，扣除积分，创建订单

### 积分抵扣
- 积分只能抵扣商品价格的 20%
- 剩余 80% 需要现金支付
- 积分价值：100 积分 = 2 元
- 积分有效期：12 个月
