# 绿行积分用户端访问解决方案

## 📱 问题诊断

您提到的用户端地址 `http://192.168.10.17:8000/user.html` 无法访问，可能原因：

1. **本地服务器未启动** - 192.168.10.17 是局域网IP，需要本地服务器运行
2. **IP地址已变更** - 电脑重启后IP可能改变
3. **手机和电脑不在同一网络** - 需要连接同一个WiFi

---

## ✅ 解决方案

### 方案1：获取当前正确的IP地址

在Mac电脑上执行以下命令：

```bash
# 查看本机IP地址
ifconfig | grep "inet " | grep -v 127.0.0.1

# 或者
ipconfig getifaddr en0
```

### 方案2：启动本地服务器

在项目根目录执行：

```bash
cd /Users/apple/WorkBuddy/20260310225816

# 使用Python启动服务器
python3 -m http.server 8000 --bind 0.0.0.0

# 或使用Node.js
npx http-server -p 8000 -a 0.0.0.0
```

### 方案3：直接打开本地文件（仅限测试）

如果只需要本地测试，可以直接在浏览器打开：
```
file:///Users/apple/WorkBuddy/20260310225816/user-app.html
```

## ⚠️ 常见问题

### Q1: 提示"无法访问此网站"
- 检查防火墙设置
- 确保手机和电脑在同一网络
- 尝试关闭电脑防火墙临时测试

### Q2: 页面加载但功能不正常
- 允许浏览器定位权限（GPS功能需要）
- 清除浏览器缓存后重试

### Q3: 需要外网访问
- 使用内网穿透工具（如ngrok）
- 或部署到云服务器
