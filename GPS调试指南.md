# 🚲 常骑骑GPS设备调试指南

## 一、硬件接线说明

### 博实结KG12 GPS定位器4线接线（您的设备）

| 线颜色 | 功能 | 说明 |
|--------|------|------|
| **红色** | 电源正极 (VCC) | 接电瓶电源正极 (12V/24V常电) |
| **黑色** | 电源负极 (GND) | 接电瓶电源负极/搭铁 |
| **绿色** | 油泵电路控制 | 接断油断电继电器（可选功能） |
| **黄色/橙色** | 继电器控制 | 远程断油电控制线（可选） |

### 接线示意图
```
电瓶车电瓶/整车线路:
┌─────────────────┐
│   正极 (12V/24V) ──┼──→ 红色线 (VCC 电源正极)
│                    │
│   负极/搭铁 ──────┼──→ 黑色线 (GND 电源负极)
│                    │
│   ACC钥匙线 ───────┼──→ 白色线 (ACC 开关检测)
│                    │
│   油路/继电器 ────┼──→ 黄色线 (断油电控制，可选)
└─────────────────┘
```

### 电瓶车接线提示（您的设备）
- **红色线**：接电瓶正极（整车电源主线，12V/24V）
- **黑色线**：接电瓶负极或车架金属部分（搭铁）
- **绿色线**：接断油断电继电器负极（远程断电功能）
- **黄色/橙色线**：接断油断电继电器正极（远程断电功能）

### 接线示意图（您的设备）
```
电瓶车电瓶/整车线路:
┌─────────────────┐
│   正极 (12V/24V) ──┼──→ 红色线 (VCC 电源正极)
│                    │

│                    │

└─────────────────┘
```

## 二、调试步骤（博实结KG12）

### 第一步：基础检查
1. **检查SIM卡**
   - 确保SIM卡已正确插入设备卡槽
   - 检查SIM卡是否有流量（4G卡）
   - 确认SIM卡已激活

2. **检查指示灯**（设备正常工作时）
   - **电源灯（红色）**：常亮 = 正常供电
   - **GPS灯（蓝色）**：闪烁 = 定位成功
   - **4G/信号灯（绿色）**：常亮 = 网络已连接

### 第二步：设备激活（博实结APP）
1. 扫描设备机身二维码，或输入IMEI号
2. 绑定设备到您的账户
3. 在APP中查看设备状态
4. 确认设备已上线

### 第三步：功能测试
| 功能 | 测试方法 |
|------|----------|
| 定位测试 | 在空旷地带等待3-5分钟，检查位置 |
| 信号测试 | 检查4G信号强度（APP显示） |
| 轨迹测试 | 移动车辆，观察APP轨迹是否正常 |
| 里程测试 | 骑行一段距离，对比实际里程 |

## 三、常见问题排查

### 问题1：设备不在线
| 可能原因 | 解决方法 |
|----------|----------|
| SIM卡欠费 | 充值流量 |
| 信号不好 | 移到信号好的地方 |
| 电源未接好 | 检查接线是否牢固 |

### 问题2：定位不准
| 可能原因 | 解决方法 |
|----------|----------|
| GPS天线被遮挡 | 调整设备位置 |
| 首次定位等待时间不足 | 等待3-5分钟 |
| 环境信号干扰 | 移到开阔地带 |

### 问题3：设备无响应
| 可能原因 | 解决方法 |
|----------|----------|
| 电源问题 | 检查红色正极、黑色负极接线 |
| 保险丝烧断 | 检查电瓶车保险丝 |
| 设备损坏 | 联系商家售后 |

## 四、与平台对接

### 需要获取的信息（联系设备商）
1. **服务器地址** - 您的服务器IP或域名
2. **端口号** - TCP端口号
3. **设备ID** - SIM卡号或IMEI号
4. **通信协议** - JT/T 808 或自定义协议

### 数据接收后
平台将解析以下数据：
- 经纬度位置
- 速度
- 方向/航向
- 时间戳
- 电池电压（如有）
