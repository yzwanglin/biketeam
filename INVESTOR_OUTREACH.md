## 四、接触流程与话术

### 4.1 第一次接触

#### 目标
- 建立初步印象
- 获取进一步沟通机会

#### 话术模板
```
"XX老师您好，我是绿行积分创始人张三。我们做了一个骑行积分平台，
用户骑行获得积分，可以抵扣商城购物。市场有3.5亿电瓶车，
千亿级市场。我们的单车月利润320元，投资回报2.5个月。
希望能占用您10分钟时间，给您详细介绍下。"
```

### 4.2 第一次会议（约20-30分钟）

#### 目标
- 详细介绍项目
- 解答投资人疑问
- 争取第二次深入交流

#### 会议准备
- 准备PPT（20页）
- 准备演示demo
- 准备数据支撑材料
- 准备竞品分析

#### 关键问题应对
| 问题 | 标准答案 |
|------|---------|
| 为什么做这个项目？ | 市场空白+政策红利+用户刚需 |
| 竞争对手会进来吗？ | 先发优势+生态闭环+数据壁垒 |
| 用户为什么愿意用？ | 积分激励+免费硬件+赚钱机会 |
| 怎么获取第一批用户？ | 免费GPS+外卖平台合作+地推 |
| 单车320元利润怎么来的？ | 广告300+佣金50+会员10=360元 |

### 4.3 第二次会议（约1-2小时）

#### 目标
- 深入沟通细节
- 展示团队实力
- 谈判投资条款

#### 会议准备
- 详细的财务预测
- 团队详细介绍
- 产品原型演示
- 用户反馈数据

### 4.4 谈判与签约

#### 谈判要点
- 估值：3000-5000万
- 股权：10-15%
- 投资条件：
  - 对赌条款：尽量拒绝
  - 董事席位：可以给1个
  - 优先清算：标准条款
  - 反稀释：可以接受

#### 签约前准备
- 尽职调查材料准备
- 法律文件审核
- 公司章程修改
- 股权变更登记

## 五、常见问题与应对

### 5.1 市场相关

#### Q：市场有多大？
```
A：3.5亿电瓶车，年骑行2000亿公里，8000亿广告市场。
即使只渗透1%，也有350万辆，月营收1亿，年营收12亿。
```

#### Q：为什么现在做？
```
A：政策红利（碳中和）、技术成熟（5G+物联网）、
消费升级（环保意识）、市场空白（国内尚无）、
资本关注（ESG投资）。
```

### 5.2 产品相关

#### Q：用户为什么愿意安装？
```
A：免费GPS定位器+骑行赚钱+积分抵现+20%现金抵扣，
相当于骑行2公里就可以买一瓶可乐，用户有强烈动机。
```

#### Q：硬件被盗怎么办？
```
A：GPS定位器有断电报警，实时追踪，而且我们提供免费更换
（用户需支付押金）。广告屏也是防盗设计。
```

### 5.3 竞争相关

#### Q：哈啰、美团会不会进来？
```
A：我们做的是差异化市场。哈啰是共享单车，不支持私家车。
美团是外卖+共享单车，没有积分体系。我们占领的是私家车
市场，这是他们没有的。
```

#### Q：如果大厂进来怎么办？
```
A：先发优势（已经占领市场）、数据壁垒（用户画像+行为数据）、
生态闭环（骑行-积分-商城-广告）、品牌认知。即使大厂进来，
我们已经建立壁垒。
```

### 5.4 盈利相关

#### Q：广告怎么卖？
```
A：按次收费（0.1元/次）+包月套餐（300元/月）。
我们有高频触达（日均曝光5-10次），精准投放（基于位置和用户画像），
广告主愿意付费。
```

#### Q：商城商品哪里来？
```
A：与电商平台合作（天猫、京东），直接接入SKU。
用户使用积分抵扣，我们从中抽成5-10%。
```

### 5.5 团队相关

#### Q：团队能力如何？
```
A：CEO前阿里P8，10年互联网经验；CTO前腾讯T9，12年技术经验；
COO前美团区域总监，8年运营经验。互补性强，执行力强。
```

#### Q：为什么离职创业？
```
A：看到了巨大的市场机会，希望做一件对社会有价值的、
有影响力的事情。
```

## 八、注意事项

### 8.1 一定要避免的错误

❌ **不要只发送BP，没有正文**
- 投资人每天收到100封BP，没有正文会直接忽略

❌ **不要群发邮件**
- 看起来像垃圾邮件，会被直接删除

❌ **不要夸大数据**
- 尽职调查会查出来，信用一旦丧失很难挽回

❌ **不要贬低竞争对手**
- 会显得格局小，显得你不懂竞品

❌ **不要拒绝反馈**
- 投资人的建议很有价值，即使不投也能学习

### 8.2 一定要做到的事情

✅ **保持诚实**
- 不知道就是不知道，不要胡编乱造

✅ **展现激情**
- 投资人投的是人，要展现你对项目的热爱

✅ **数据支撑**
- 每一个观点都要有数据支撑

✅ **持续跟进**
- 不主动跟进的投资人不会投你

✅ **建立口碑**
- 即使不投，也要保持良好关系
