# E-Bike Manager (Web 版本)

## ✅ 已完成！

Web 版本的 E-Bike Manager 已经创建完成，可以直接在浏览器中运行！

## 📱 功能特性

### 1. 仪表盘
- 实时统计：在线/离线车辆、活跃广告、已获礼包
- 当前车辆详细信息展示
- 数据可视化卡片

### 2. 车辆管理
- 添加新车辆
- 删除车辆
- 选择当前车辆
- 查看车辆状态（GPS、速度、电量）

### 3. 骑行地图
- 集成 OpenStreetMap 地图
- 实时显示车辆位置
- 开始/停止骑行模式
- 骑行时模拟位置移动和速度变化
- 点击礼包图标可领取奖励

### 4. 广告管理
- 创建新广告
- 查看广告列表
- 显示广告状态（投放中/已暂停）

### 5. 骑行游戏
- 礼包地图显示
- 随机放置礼包
- 收集礼包（优惠券、积分）
- 礼包统计

## 📖 使用说明

### 仪表盘
- 查看所有统计信息
- 当前车辆详情在底部卡片中显示

### 车辆管理
- 点击"添加车辆"创建新车
- 点击"选择"按钮切换当前车辆
- 点击"删除"按钮移除车辆

### 骑行地图
- 点击"开始骑行"按钮开启骑行模式
- 车辆会每 3 秒自动移动并更新速度
- 点击地图上的 🎁 图标可以领取礼包
- 点击"放置礼包"可以在地图上随机添加礼包

### 广告管理
- 点击"添加广告"创建新广告
- 查看所有广告列表

### 骑行游戏
- 查看礼包统计（可用、已收集、优惠券、积分）
- 查看已收集的礼包列表

## 💡 技术特点

- **纯 HTML/CSS/JavaScript** - 无需构建工具
- **Leaflet.js** - 开源地图库
- **响应式设计** - 支持不同屏幕尺寸
- **单文件架构** - 所有代码在一个 HTML 文件中
- **本地存储** - 使用内存存储（刷新页面数据重置）
